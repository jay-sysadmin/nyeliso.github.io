02 // Technology Stack

What We Used & Why

🐄 Mailcow

Dockerized all-in-one mail server — includes Postfix, Dovecot, SOGo, ClamAV, Rspamd, and more. Fast, reliable, web-managed.

🐧 Linux Mint

Server OS — free, stable, open-source. Eliminates licensing costs and keeps the project within the N$100,000 budget.

🌐 SOGo Webmail

Browser-based email client included in Mailcow. Employees access email from anywhere — no software install needed.

🔒 WireGuard VPN

Secure VPN tunnel allowing remote employees to access the mail server safely from outside the office network.

🦠 ClamAV

Open-source antivirus integrated in Mailcow — scans all incoming and outgoing emails for viruses and infected attachments.

🛡️ Rspamd + SpamAssassin

Anti-spam filtering — blocks junk mail before it reaches user inboxes. Integrated directly into the Mailcow stack.

🔐 TLS Encryption

TLS 1.3 enabled for SMTP, POP3, and IMAP — encrypts all emails in transit. HTTPS secures webmail access.

🗂️ Postfix + Dovecot

Postfix handles outgoing mail (SMTP). Dovecot handles incoming mail storage and retrieval (IMAP/POP3).

04 // Setup Process

Step-by-Step Implementation

Set up Linux Mint as the server OS. Installed Docker and Docker Compose — required to run Mailcow's containerized stack.

Cloned the Mailcow repository, configured the mailcow.conf file with the hostname (mail.earnmorecc.local) and timezone (Africa/Windhoek), then launched all containers.

root@mail:~# cd mailcow-dockerized && ./generate_config.sh

    Company Policy Implemented

    📵 No Personal Use

    Email system is strictly for business use. Personal use is prohibited and may result in disciplinary action.

    ⚖️ Legal Disclaimers

    A standard legal disclaimer is automatically appended to all outgoing emails via Postfix/altermime.

    🏢 earnmorecc.com Domain

    All accounts follow the format name@earnmorecc.com for consistent company branding.

    🚫 No Offensive Content

    Email content must not include offensive, discriminatory, or disruptive messaging — violations result in disciplinary action.

    💰 Budget N$100,000

    Met by using Linux Mint (free) and Mailcow (open-source) — zero software licensing costs.

    💾 5–10 GB Per Mailbox

    Each employee mailbox allocated 5–10 GB depending on email volume and retention policy.
