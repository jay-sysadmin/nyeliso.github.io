010 // Lab Report

Static Routing — Dual-Router Topology

Built a multi-network topology with two routers interconnected over a WAN link. Configured static routes on both routers to enable end-to-end communication between two separate LANs — simulating a real-world site-to-site routing scenario.

Cisco Packet Tracer Cisco IOS CLI 2× Router 2911 2× Switch 2960 Networking Lab
01 // Objective
Configure static routes on two interconnected routers so that PC0 on Network A (192.168.10.0/24) can communicate with PC1 on Network B (192.168.20.0/24) through a WAN link (10.0.0.0/30).

03 // Configuration
Router1 — Interfaces + Static Route to Network B
enable
conf t

interface g0/0
ip address 192.168.10.1 255.255.255.0
no shutdown
exit

interface g0/1
ip address 10.0.0.1 255.255.255.252
no shutdown
exit
! Static route: to reach Network B, send via Router2 WAN IP
ip route 192.168.20.0 255.255.255.0 10.0.0.2
Router2 — Interfaces + Static Route to Network A
enable
conf t

interface g0/0
ip address 10.0.0.2 255.255.255.252
no shutdown
exit

interface g0/1
ip address 192.168.20.1 255.255.255.0
no shutdown
exit
! Static route: to reach Network A, send via Router1 WAN IP
ip route 192.168.10.0 255.255.255.0 10.0.0.1
