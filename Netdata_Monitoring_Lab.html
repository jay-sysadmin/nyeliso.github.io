01 // Overview

What is Netdata?

Definition Netdata is a free, open-source real-time monitoring tool for Linux systems. It tracks thousands of system metrics instantly — no delays — and displays them on a live browser-based dashboard. Founded in 2013 by Costa Tsaousis, owned by Netdata Inc.
  • Shows live data instantly — unlike other tools that delay updates, you see problems as they happen
  • Monitors thousands of metrics automatically — CPU, RAM, disk, network, processes, and more, all at once
  • No special software needed — just open a browser and go to http://[server-ip]:19999
  • Organises metrics into sections (CPU, RAM, Disk, Network) so you always know where to look
  • Includes AI Insights, anomaly detection, and configurable alerts

📊 Real-Time Graphs

Live charts update every second — disk reads/writes, CPU load, RAM usage, network traffic.

🔔 Alerts System

Raises warnings and critical alerts automatically — e.g. reboot required after package updates.

🤖 AI Insights

Built-in anomaly detection flags unusual behaviour in your metrics automatically.

🌐 Browser Dashboard

No client software needed — access from any browser on the same network.

🖥️ Nodes Explorer

View all monitored servers in one place — status, uptime, CPU, RAM, load, disk.

📁 Organised Metrics

Grouped into: System, Compute, Memory, Storage, Network, Hardware, Processes, Apps.

02 // Installation & Setup

How I Installed Netdata

1
Prepared the Linux Ubuntu VM
Running Ubuntu on a virtual machine named linuxjay (user: jay23). Confirmed internet connectivity and updated packages before installing.
2
Installed Netdata via the official kickstart script
Used the one-line installer from netdata.cloud — it automatically detects the OS, installs dependencies, and sets up Netdata as a systemd service.
jay23@linuxjay: ~
jay23@linuxjay:~$ wget -O /tmp/netdata-kickstart.sh https://get.netdata.cloud/kickstart.sh
jay23@linuxjay:~$ sh /tmp/netdata-kickstart.sh
  • ✓ Netdata installed successfully
    • jay23@linuxjay:~$ sudo systemctl enable netdata
    jay23@linuxjay:~$ sudo systemctl start netdata
    jay23@linuxjay:~$ sudo systemctl status netdata
